# Бэклог задач

## Как использовать

- Каждая задача должна давать конкретный проверяемый результат.
- Статус задачи меняется в этом файле, чтобы код и план не расходились.
- Крупные задачи дробятся только до того уровня, на котором их реально можно закрыть за один проход.
