# intern-assignment-03

## Описание

Тестовое задание на Kotlin Multiplatform: экран каталога продуктов с поиском,
фильтрацией по категориям, сортировкой и пагинацией.

## Текущий статус

Проект инициализирован:
- подготовлена KMP-структура с модулями `model`, `feature`, `app`
- подключены базовые зависимости под задание
- `app` связан с shared-модулем `feature`
- добавлен стартовый placeholder-экран из shared-слоя
- документация в `docs/` приведена в рабочее состояние

## Стек

- Kotlin Multiplatform
- Jetpack Compose
- Koin
- Kotlinx Coroutines
- Kotlinx Serialization
- Ktor Client

1. Открыть проект в Android Studio
